# Wealthify Frontend

Simple React app for user authentication.

## Files:
- `src/App.js` - Main app component (controls login/signup/dashboard views)
- `src/App.css` - All styling
- `src/index.js` - Entry point (renders App)
- `src/index.css` - Basic body styles
- `src/components/Login.js` - Login form
- `src/components/Signup.js` - Signup form
- `src/components/Dashboard.js` - User dashboard after login

## To run:
```bash
npm start
```

Runs on http://localhost:3000
